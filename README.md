# FastVPN - Бесплатное расширение для VPN и прокси

## Описание
FastVPN — это полноценное браузерное расширение для Chrome/Edge, которое автоматически собирает актуальные бесплатные V2Ray (VLESS, VMess и др.) конфигурации из открытых источников на GitHub. 

Подходит для клиентов: V2RayN, Nekobox, Hiddify, Happ, V2RunTun и других.

## Функции
- Автоматический поиск и обновление рабочих прокси/нод
- Генерация subscription ссылок
- Простой интерфейс
- Работает без сбоев для обхода ограничений

## Установка
1. Скачайте ZIP архив
2. Распакуйте
3. В Chrome: chrome://extensions/
4. Включите "Режим разработчика"
5. "Загрузить распакованное расширение" → выберите папку

## Как использовать
- Нажмите на иконку расширения
- "Получить свежие подписки"
- Скопируйте или скачайте подписку и импортируйте в VPN-клиент

## Источники
Скрипт агрегирует из популярных GitHub репозиториев с бесплатными конфигами.

## Предупреждение
Бесплатные ноды могут быть нестабильными. Используйте на свой страх и риск. Для максимальной надежности рекомендуется собственный сервер.
